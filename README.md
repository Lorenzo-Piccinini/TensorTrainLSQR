# TensorTrainLSQR
Codes for the paper "TT-LSQR for tensor least squares problems and application to data mining", by L. Piccinini and V. Simoncini. 
DOI: 10.1007/s11075-025-02204-8

In order to properly run the algorithm in this repository, be sure to have first downloaded the Tensor Train (TT) matlab toolboc by Oseledets, https://github.com/oseledets/TT-Toolbox

TT_Tensorized_LSQR.m is the main algorithm: LSQR is implemented using TT format to represent tensors. In the repository all the codes to reproduce the experiments in the article are available.
